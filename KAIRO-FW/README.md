# KAIRO-FW

KAIRO-FW is the KAIRO ecosystem packaging surface for an AI-driven network perimeter and firewall component.

## Debian package

The packaged baseline for version `0.2.0` is staged under `KAIRO-FW/deb/`.

This repository snapshot does not build or install runtime enforcement binaries in the `.deb`; it packages documentation and metadata only as a baseline.
